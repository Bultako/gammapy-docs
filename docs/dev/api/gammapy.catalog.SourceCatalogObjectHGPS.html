spatial_model(emin=<Quantity 1. TeV>, emax=<Quantity 100000. TeV>)[source]¶

Spatial model (SpatialModel).

One of the following models (given by Spatial_Model in the catalog):

  • Point-Like or has a size upper limit : Delta2D
  • Shell: Shell2D
  • Gaussian: gammapy.image.models.Gaussian2D
  • 2-Gaussian or 3-Gaussian: composite model (using + with Gaussians)
spectral_model(which='best')[source]¶

  • ECPL : ExponentialCutoffPowerLaw
