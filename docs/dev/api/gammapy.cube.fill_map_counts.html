fill_map_counts¶

gammapy.cube.fill_map_counts(count_map, event_list)[source]¶

Fill events into a counts map.

The energy of the events is used for a non-spatial axis homogeneous to energy. The other non-spatial axis names should have an entry in the column names of the EventList

Parameters:

count_map : Map

Map object, will be filled by this function.

event_list : EventList

Event list
