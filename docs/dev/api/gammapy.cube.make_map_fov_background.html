make_map_fov_background¶

gammapy.cube.make_map_fov_background(acceptance_map, counts_map, exclusion_mask)[source]¶

Build Normalized background map from a given acceptance map and count map.

This operation is normally performed on single observation maps. An exclusion map is used to avoid using regions with significant gamma-ray emission. All maps are assumed to follow the same WcsGeom.

TODO: A model map could be used instead of an exclusion mask.

Parameters:

acceptance_map : WcsNDMap

Observation hadron acceptance map (i.e. predicted background map)

counts_map : WcsNDMap

Observation counts map

exclusion_mask : WcsNDMap

Exclusion mask
Returns:

norm_bkg_map : WcsNDMap

Normalized background
